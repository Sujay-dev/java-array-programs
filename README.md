* Reverse Array
* Move Zeros to End
* Missing Number
* Frequency of Elements
* Prefix Sum
* Array Rotation (Left / Right / K Rotations)

### Searching Algorithms

* Linear Search
* Binary Search

### Matrix Problems

* Matrix Addition
* Transpose Matrix
* Spiral Matrix
* Column Wise Sum
* Maximum Element in Matrix
* Non-Square Matrix Transpose

### String Problems

* Reverse String
* Count Vowels
* Anagram Check

### Mini Projects

* Player Performance Analyzer

* Problem Solving
